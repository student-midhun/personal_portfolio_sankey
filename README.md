Personal Portfolio Website

This repository contains a personal portfolio website built as part of a frontend assignment.
The project is developed using HTML, CSS, and JavaScript only, without using any external frameworks or libraries.

The portfolio demonstrates a clean, responsive layout along with basic interactive features, making it suitable for academic submission and beginner-level frontend development showcase.

---

Features

* Responsive layout for different screen sizes
* Light and Dark theme toggle
* Animated text in the About section
* Structured sections including About, Projects, Achievements, and Contact
* Social media icons linked to external profiles
* Clean user interface with hover effects and smooth transitions
* Deployed using GitHub Pages

---

Technologies Used

* HTML5 for page structure and content
* CSS3 for styling, layout, responsiveness, and transitions
* JavaScript for theme switching, animated text, and interactivity

No frameworks, libraries, or templates were used in this project.

Light / Dark Mode

The website includes a light and dark theme toggle feature.
The selected theme is stored using localStorage so that the preference remains saved even after refreshing or reopening the website.

---

Deployment

The project is deployed using GitHub Pages.

To view the live site:

* Open the repository settings
* Navigate to Pages
* Select the main branch and root directory

Live website URL:
[https://student-midhun.github.io/personal_portfolio_sankey/](https://student-midhun.github.io/personal_portfolio_sankey/)

How to Run Locally

1. Clone the repository using:
   git clone [https://github.com/student-midhun/personal_portfolio_sankey.git](https://github.com/student-midhun/personal_portfolio_sankey.git)

3. Open index.html in any modern web browser
